#### Clone this repository

````
$ git clone https://github.com/Blinina/frontend-project-lvl2.git
````
#### Setting

````
$ make install
$ npm link

````
#### Display help information
```
$ gendiff -h

```
#### Using exampl

```
$ gendiff file1.json file2.json
```
